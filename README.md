Nihonbyte adalah aplikasi web offline yang saya buat sendiri untuk belajar kosakata bahasa Jepang, ditujukan khusus untuk pemula hingga menengah (level JLPT N5–N4).<br>
Aplikasi ini menyajikan materi dalam bentuk kartu kata interaktif, poster hiragana/katakana, pola kalimat dasar, serta dilengkapi audio pengucapan dan rekomendasi kata terkait.

<br>
Semua dibangun dengan HTML, CSS, dan JavaScript vanilla — sengaja dibuat ringan, tanpa framework berat, sehingga bisa langsung dibuka di browser tanpa instalasi atau koneksi internet.<br>
Cocok untuk belajar santai di laptop maupun HP.

## Fitur Utama

- **Kosakata Lengkap**: Lebih dari 1400 kata dan ungkapan JLPT N5–N4, dikelompokkan berdasarkan jenis (kata benda, kata kerja, kata sifat, ekspresi, dll.).
- **Kartu Interaktif**: Menampilkan kanji, kana, romaji, dan arti dalam bahasa Indonesia, lengkap dengan tombol audio pengucapan.
- **Modal Detail**: Klik kartu untuk tampilan diperbesar + rekomendasi kata sejenis.
- **Poster Hiragana & Katakana**: Tabel lengkap gojūon, dakuon, handakuon, dan yōon.
- **Pola Kalimat**: Koleksi pola dasar per level (N5–N4 saat ini).
- **Ungkapan Umum**: Tampilan khusus lebar untuk ekspresi sehari-hari.
- **Pencarian & Filter**: Cari kata berdasarkan kanji/kana/arti, filter berdasarkan kategori dan level.
- **Sidebar Navigasi**: Menu collapsible, ramah mobile (drawer mode).
- **Mode Test**: Latihan kuis sederhana (sedang dikembangkan untuk level lebih tinggi).
- **100% Offline & Ringan**: Semua aset lokal, tanpa iklan atau tracking.

<br>

## Cara Menjalankan

1. Download atau clone repository ini.<br>
2. Buka file `index.html` langsung di browser (Chrome/Edge/Firefox/Safari direkomendasikan).<br>
3. Tidak perlu server atau instalasi apa pun.

<br>

## Teknologi yang Digunakan

- HTML5<br>
- CSS3 (Grid, Flexbox, animasi ringan)<br>
- JavaScript Vanilla (ES6+)<br>
- Web Speech API (untuk audio pengucapan)<br>
- Data kosakata dan pola kalimat dalam file JSON terpisah

<br>

## Catatan Pengembangan

- Level JLPT N3–N1 (pola kalimat & test) masih dalam proses pengembangan.<br>
- Aplikasi dioptimalkan untuk browser modern dan responsif di mobile.<br>
- Audio menggunakan voice Jepang bawaan browser (hasil terbaik di Chrome/Edge).
